# Asteroids Clone

A small educational Asteroids-style game built with Python and Pygame.

## About

In this game, the player controls a spaceship, shoots asteroids, and avoids collisions.  
Large asteroids split into smaller ones, which move faster and in new random directions.

## Features

- player movement
- shooting mechanics
- collision detection
- asteroid splitting
- simple game loop with Pygame

## Tech Stack

- Python
- Pygame
